# GovData Template-Engine

A NextJS/React based frontend for GovData

## Getting Started

go to webapp folder

```bash
cd webapp
```

installing all dependencies

```bash
npm install
```

running development server on http://localhost:3000

```bash
npm run dev
```

see package.json for more scripts
